# Front-schlack

Front-schlack is a Angular Front-end Slack like

## Installation

Use the Javascript package manager [npm](https://www.npmjs.com/) to install Front-schlack.

```bash
npm install
```

## Usage

```python
# npm start or ng serve
to launch Front-schlack

# npm test or ng test
to launch the tests

```
